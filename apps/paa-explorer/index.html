Usage

Installation

# Clone the repository
git clone https://github.com/voidseo/paa-explorer.git
cd paa-explorer

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Add your OpenAI API key to .env

Basic Usage

from paa_explorer import PAAExplorer

# Initialize explorer
explorer = PAAExplorer(
    locale='en-US',
    openai_api_key='your-key'
)

# Scrape PAA questions
questions = explorer.scrape_paa([
    'seo tools',
    'keyword research',
    'content optimization'
])

# Cluster questions
clusters = explorer.cluster_questions(questions)

# Export results
explorer.export_csv(clusters, 'paa_results.csv')

Configuration

# config.yaml
scraping:
  max_questions_per_keyword: 10
  delay_between_requests: 2
  max_retries: 3
  
clustering:
  algorithm: 'kmeans'
  n_clusters: 'auto'
  min_cluster_size: 5
  
export:
  include_metadata: true
  format: 'csv'

Advanced Features

# Trend analysis
trends = explorer.analyze_trends(
    keywords=['seo tools'],
    time_range='6m',
    frequency='weekly'
)

# Custom clustering
clusters = explorer.cluster_questions(
    questions,
    algorithm='dbscan',
    min_samples=3,
    eps=0.3
)

# Export to Google Sheets
explorer.export_sheets(
    clusters,
    spreadsheet_id='your-sheet-id',
    worksheet='PAA Analysis'
)

API Reference

Core Classes

  • PAAExplorer - Main interface for scraping and clustering
  • Scraper - Handles PAA question extraction
  • Clusterer - Groups questions by similarity
  • TrendAnalyzer - Tracks changes over time

PAAExplorer.scrape_paa()

Parameters:

  • keywords (list): Seed keywords to scrape
  • locale (str): Target locale (default: 'en-US')
  • max_questions (int): Max questions per keyword

Returns: List of PAA questions with metadata

PAAExplorer.cluster_questions()

Parameters:

  • questions (list): Questions to cluster
  • algorithm (str): Clustering algorithm
  • n_clusters (int|str): Number of clusters or 'auto'

Returns: Dictionary of clustered questions
